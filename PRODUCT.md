# 基金收益预测助手 V1 产品介绍

## 1. 产品定位

基金收益预测助手 V1 是一款本地运行的基金分析与持仓管理工具，主要服务于个人投资者、课程项目展示和量化分析学习场景。系统把基金估值、历史净值、重仓股、市场指数、贵金属价格、量化买卖信号和 AI 辅助分析整合到一个轻量 Web 应用中，同时提供命令行工具用于无界面操作。

本项目不提供投资建议，所有结果仅用于辅助分析和技术研究。

## 2. 核心能力

### 2.1 持仓管理

- 支持手动添加、删除基金持仓。
- 支持文本导入持仓数据。
- 支持图片识别持仓截图，需配置 OpenAI 兼容 Vision 接口。
- 汇总展示总资产、今日估算收益、累计收益、基金数量和组合收益率。
- Web 端持仓保存在浏览器 localStorage，CLI 端持仓保存在本地 `holdings.json`。

### 2.2 基金实时分析

- 查询基金实时估值、单位净值、估算涨跌幅和更新时间。
- 查询历史净值趋势并展示图表。
- 查询近 1 月、3 月、6 月、1 年收益表现。
- 查询基金重仓股明细，用于辅助判断基金波动来源。

### 2.3 多因子买卖信号

系统使用多因子模型对基金进行评分，输出买入、观望或卖出倾向。主要因子包括：

- 均线位置：衡量当前净值相对 MA20、MA60、MA120、MA250 的趋势位置。
- RSI 指标：识别短期超买或超卖状态。
- 近期动量：结合 5 日、10 日、20 日收益变化。
- 历史回撤：评估当前净值相对阶段高点的回撤程度。
- 历史分位：判断当前净值在历史区间中的位置。

### 2.4 智能基金推荐

- 从公开基金排名数据中构建候选池。
- 结合收益能力、风险控制、夏普表现、收益一致性和技术面进行综合评分。
- 输出推荐列表及评分，适合用于进一步人工筛选。

### 2.5 市场与贵金属行情

- 展示上证指数、深证成指、创业板指等主要市场指数。
- 展示热门板块行情。
- 展示黄金、白银等贵金属价格和趋势数据。

### 2.6 AI 助手

- 支持中文投资分析对话。
- 支持流式响应。
- 支持图片识别和持仓截图解析。
- 支持从自然语言文本中提取基金代码、名称、金额和收益信息。

AI 能力依赖外部 OpenAI 兼容服务。未配置 AI Key 时，基础基金行情和量化分析功能仍可使用。

## 3. 技术架构

```text
用户层
├── Web 浏览器
└── CLI 终端

├── Flask API
├── 原生 JavaScript SPA
├── Chart.js 图表

业务层
├── 基金估值抓取

├── 重仓股查询

├── 令牌桶限流

```

## 4. 文件说明

| 文件 | 说明 |
| --- | --- |
| `app.py` | Flask 后端入口，包含 API 路由、数据抓取、AI 调用和量化分析逻辑 |
| `cli.py` | 命令行工具，复用后端能力完成持仓、信号、推荐和贵金属查询 |
| `ratelimit.py` | 令牌桶限流器，控制外部接口访问频率 |
| `config.json` | 本地运行配置，已移除真实密钥 |
| `config.example.json` | 配置模板 |
| `requirements.txt` | Python 依赖 |
| `start.bat` | Windows 一键启动脚本 |
| `templates/index.html` | Web 页面模板 |
| `static/js/app.js` | 前端主逻辑 |
| `static/css/style.css` | 前端样式 |
| `README.md` | 项目安装、配置、运行和 API 文档 |

## 5. 使用流程

1. 安装 Python 依赖。
2. 配置可选的 AI 环境变量。
3. 运行 `python app.py` 启动 Web 服务。
4. 访问 `http://localhost:5000`。
5. 添加基金持仓或导入持仓数据。
6. 查看组合收益、基金详情、信号评分和市场行情。
7. 需要终端操作时使用 `python cli.py --help` 查看命令。

## 6. V1 范围与限制

V1 版本以本地单机使用为主，重点完成完整功能闭环，而不是多人账户、云端部署或交易执行。

当前限制：

- 外部数据接口可能因网络、限流或接口变化而不可用。
- Web 持仓存储在 localStorage 中，不适合作为长期唯一备份。
- 价格提醒保存在内存中，服务重启后会清空。
- AI 能力依赖独立服务和密钥配置。
- 所有投资分析结果仅供参考。
